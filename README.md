# Indonesia Trend Intelligence

AI Dashboard untuk monitoring trend terbaru Indonesia menggunakan DeepSeek API.

## Features

- Google Trends Indonesia
- RSS News Indonesia
- AI Trend Analysis
- Trend Ranking Chart

## Installation

pip install -r requirements.txt

## Run Local

streamlit run streamlit_app.py

## Deploy

Upload ke GitHub lalu deploy di Streamlit Cloud.

## DeepSeek API

https://platform.deepseek.com/api_keys
